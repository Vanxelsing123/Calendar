# Изменения в версии 2.0

## Новый функционал

### 1. Поддержка 4 смен
- Добавлен выбор номера смены (1-4) при первой настройке
- Каждая смена работает по 4-дневному циклу: День → Ночь → Отсыпной → Выходной
- Смены смещены друг относительно друга на 1 день

### 2. Боковое меню
- Бургер-меню в левом верхнем углу
- Переключение между графиками всех 4 смен
- Просмотр графика любой бригады без потери настроек

### 3. Расширенная настройка
- Выбор начальной даты
- Выбор номера смены (1-4)
- Выбор типа смены (день/ночь) для точного старта

### 4. Новая цветовая схема
- Светло-зелёный (#6FCF97) — дневная смена
- Тёмно-фиолетовый (#5B4B8A) — ночная смена
- Красный (#E74C3C) — выходные и отсыпной
- Тёмно-жёлтый (#D4AF37) — больничные
- Светло-синий (#5DADE2) — отпуск

### 5. Улучшенный UX
- Индикатор текущей просматриваемой смены в header
- Визуальное выделение активной смены в меню
- Плавные анимации открытия меню

## Технические изменения

- Обновлена логика расчёта смен для поддержки 4 бригад
- Добавлено сохранение номера смены и типа в localStorage
- Добавлен компонент SideMenu для навигации
- Обновлена структура хранилища данных
