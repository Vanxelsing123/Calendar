# График смен

PWA-приложение для отслеживания скользящего графика работы 4 смен.

## Функционал

- **4 бригады** работают по скользящему графику
- **Автоматический расчёт** цикла: День → Ночь → Отсыпной → Выходной
- **Первоначальная настройка**: выбор даты, номера смены (1-4), типа смены (день/ночь)
- **Переключение между графиками** разных смен через боковое меню
- **Цветовая схема**:
  - Светло-зелёный — дневная смена (8:00-20:00)
  - Тёмно-фиолетовый — ночная смена (20:00-8:00)
  - Красный — выходные и отсыпной
  - Тёмно-жёлтый — больничные
  - Светло-синий — отпуск
- **Редактирование дней**: клик на день открывает модалку для изменения типа и добавления заметок
- **Вертикальная прокрутка** месяцев (±6 месяцев)
- **Сохранение в localStorage**: все данные сохраняются локально
- **PWA** для установки на телефон

## Логика смен

Каждая из 4 бригад работает по циклу:
1. День (8:00-20:00)
2. Ночь (20:00-8:00)
3. Отсыпной
4. Выходной

Бригады смещены друг относительно друга на 1 день, обеспечивая непрерывную работу.

Пример на 14-17 января:
- 14.01: Смена 1 (день), Смена 4 (ночь)
- 15.01: Смена 2 (день), Смена 1 (ночь)
- 16.01: Смена 3 (день), Смена 2 (ночь)
- 17.01: Смена 4 (день), Смена 3 (ночь)

## Установка зависимостей

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Откроется на `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
```

Результат в папке `dist/`

## Деплой

### Vercel
```bash
npm i -g vercel
vercel
```

### Netlify
```bash
npm i -g netlify-cli
netlify deploy --prod --dir=dist
```

### GitHub Pages
```bash
npm run build
# Загрузи содержимое dist/ в gh-pages ветку
```

### Любой хостинг
Просто загрузи содержимое папки `dist/` на хостинг (поддержка SPA required).

## Установка на телефон

1. Открой приложение в браузере (Chrome/Safari)
2. Меню → "Добавить на главный экран"
3. Готово
